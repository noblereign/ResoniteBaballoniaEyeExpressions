# BaballoniaEyeExpressions

A [ResoniteModLoader](https://github.com/resonite-modding-group/ResoniteModLoader) mod for [Resonite](https://resonite.com/) that adds support for Baballonia's new eye expressions.

> [!IMPORTANT]
> If this mod helped you, you should probably thumbs-up the relevant entry in the Resonite issue tracker!
>
> [**#6821** - Babble eye expressions no longer works with the new Baballonia "expressions" branch](https://github.com/Yellow-Dog-Man/Resonite-Issues/issues/6821)

## Installation
1. Install [ResoniteModLoader](https://github.com/resonite-modding-group/ResoniteModLoader).
2. Place [BaballoniaEyeExpressions.dll](https://github.com/noblereign/BaballoniaEyeExpressions/releases/latest/download/BaballoniaEyeExpressions.dll) into your `rml_mods` folder. This folder should be at `C:\Program Files (x86)\Steam\steamapps\common\Resonite\rml_mods` for a default install. You can create it if it's missing, or if you launch the game once with ResoniteModLoader installed it will create this folder for you.
3. Start the game. If you want to verify that the mod is working you can check your Resonite logs.
